# rdp-socks5
Remote Desktop over Socks5 proxy

# usage
## Run proxy
```
rdp-socks5.exe -l 127.0.0.1:3388 -r 10.0.9.11:3389 -x "socks5://127.0.0.1:1081?timeout=5m"
```
